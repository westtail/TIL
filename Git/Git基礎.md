## Gitとは
* 分散型バージョン管理システム
* ファイルの状態を好きなときに更新履歴として保存できる
* 更新履歴から特定のバージョンに戻したり、更新するバージョンを指定できる

## 基本用語

* リポジトリ
  * 過去ソースコードのデータベース 履歴などの保存場所
* ローカルリポジトリ
  * 各PCにあるデータベース、リポジトリ　gitコマンドを基本的に行う場所
* リモートリポジトリ
  * 各PC以外の保存用のリポジトリ　ローカルリポジトリのバックアップ
  * GitHubやBitbucketなどが有名
